# lox.ts

lox.ts is an implementation of the Lox programming 
in Typescript.

## Setup:

- Ensure Bun is installed.
  - Ref : [Bun Install](https://bun.sh/docs/installation)
- To install dependencies:

```bash
bun install
```

To run:

```bash
bun run dev
```

Generate a single CLI binary as:
```bash
bun run build
```
